# CraftZone Nexus

Fabric server mod for Minecraft `1.21.11` with:
- Economy and shop systems
- Claims and clans
- Bounty and wanted systems
- Anti-cheat checks
- Discord logging and dashboard features

## Compatibility
- Minecraft: `1.21.11`
- Fabric Loader: `0.18.4+`
- Java: `21`

## Build
```powershell
./gradlew build
```

Release jar:
- `build/libs/core-mod-<version>.jar`

## Config
- Main config: `config/core-mod.json`
